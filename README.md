# Git insights

A tool for analyzing git repositories.

Runs with `uv`.

Set up with: 

```
uv sync
source .venv/bin/activate
```

Run with:

```
uv run git_analyzer/main.py <path-to-repo>
```
textual run --dev start.py <path-to-git-repo>
```

Then in another terminal:

```
textual console
```

For debug logs.
